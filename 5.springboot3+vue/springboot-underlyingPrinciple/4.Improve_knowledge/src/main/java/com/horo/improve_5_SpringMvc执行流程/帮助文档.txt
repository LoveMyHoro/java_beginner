1.ModelAndView是 Spring MVC 用于在控制器（Controller）和视图（View）之间传递数据并指定视图的一个重要类，
以下是关于它的详细介绍：
    (1)数据传递：ModelAndView充当了一个数据容器的角色，控制器可以将需要在视图中展示的数据放入其中。这些数据可以是
    各种类型，比如简单的字符串、数字，也可以是复杂的 Java 对象（如实体类、集合等）。例如，在一个处理用户订单信息的
    控制器方法中，可以将订单对象、用户信息等相关数据放入ModelAndView中，以便视图能够获取并展示这些内容。
    (2)视图指定：除了传递数据，ModelAndView还用于指定要渲染的视图。它可以明确地告诉 Spring MVC，在处理完控制器
    方法后，应该使用哪个视图来呈现最终的结果。比如，在一个 Web 应用中，可能有多个不同的 JSP 页面作为视图，通过
    ModelAndView就可以确定是要渲染 index.jsp、userInfo.jsp还是其他的页面。
